# IT Support Ticket System

## Overview
A command-line IT support ticket system built using Python and SQLite. This project simulates real-world help desk workflows.

## Features
- Create, view, and update tickets
- Search tickets by keyword
- Persistent database storage (SQLite)
- Logging system for tracking actions and errors
- Error handling for stability

## Technologies
- Python
- SQLite
- Git & GitHub

## How to Run
```bash
python main.py
